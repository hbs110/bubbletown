bubbletown
==========

**调试和启动步骤**

1. 下载和解压缩 cocos2d-x 3.3 完全版
2. 将本 repo 内的内容 clone 至 {root_path}/games/ 目录下。
3. 如果使用 Visual Studio，将 bubble-town 的 Project Settings 的 Debugging 内的 Working Directory 设置为 “$(ProjectDir)../Resources” 即可启动调试
